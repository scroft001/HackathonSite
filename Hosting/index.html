Why Scratch?

Scratch is a very basic visual blocks editor used to create video games. It is super easy to learn and start building interactive games that will run anywhere in a browser. Why is this a good program to use at an arcade night? Most schools that have a "coding club" use Scratch to introduce their students to coding. At Hackingtons we use it as a stepping stone for the younger students to start learning the basics of code. It is widely known in the education sector meaning most kids will have heard of it. It is wildly popular which increases the turn out as most kids know Scratch and can build "a totally awesome game that will make me a lot of money".
